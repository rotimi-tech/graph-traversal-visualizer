# Graph Traversal Visualizer | Java

## Overview
This project implements graph traversal algorithms using Java and demonstrates how BFS and DFS explore connected nodes.

## Technologies Used
- Java
- BlueJ
- Graph Data Structures

## Features
- Breadth-First Search
- Depth-First Search
- Graph Visualization
- Node and Edge Implementation

## Skills Demonstrated

- Object-Oriented Programming
- Debugging
- Data Structures
